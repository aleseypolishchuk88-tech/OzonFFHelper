# OzonFFHelper

Программа для подготовки поставок Ozon FBO и работы с фулфилмент-компанией (ФФ):
- собирает **листы «Оприходование»** (что отправляем ФФ) из состава поставок;
- по **описи от ФФ** создаёт **грузоместа** в Ozon и выдаёт **ZIP с этикетками**, переименованными в PAC-коды.

Программа работает **локально на вашем компьютере**. Это не сайт в интернете: она просто открывается в окне браузера по адресу вашего же компьютера. Данные никуда не загружаются.

---

## 1. Что нужно один раз установить

1. **Python** — нужен «классический» установщик, иначе программа не запустится.
   - Откройте https://www.python.org/downloads/windows/
   - В разделе **«Stable Releases»** найдите **Python 3.12.x** и скачайте **«Windows installer (64-bit)»** — обычный `.exe`.
   - ⚠️ НЕ ставьте вариант **«Python install manager»** — с ним программа работать не будет.
   - Запустите, на первом экране поставьте галочку **«Add python.exe to PATH»**, затем «Install Now».
2. Распакуйте папку **OzonFFHelper** в удобное место (например, на диск D).

---

## 2. Запуск

1. Откройте папку OzonFFHelper и дважды кликните по файлу **`start.bat`**.
2. Первый запуск идёт ~3 минуты — программа сама ставит нужные библиотеки. Дальше запускается за секунды.
3. Откроется (или нужно открыть вручную) браузер по адресу **http://localhost:8501**.
4. Чтобы закрыть программу — просто закройте чёрное окно, которое открылось вместе с ней.

---

## 3. Первый вход — ключи Ozon

При первом запуске программа попросит **Client-Id** и **Api-Key**.
- Возьмите их в личном кабинете Ozon: **Настройки → Seller API → сгенерировать ключ**.
- Вставьте оба значения и нажмите **«Проверить и сохранить»**. Ключи сохранятся на вашем компьютере, больше вводить не нужно.

> Каждый менеджер использует **свой ключ**. Сменить ключ можно в боковой панели → «Сменить ключи».

**Роли ключа.** При создании ключа выберите уровень доступа. Чтобы работало всё, включая авто-выравнивание состава заявки под опись, ключу нужна роль **«Supply order»** (полная, не ReadOnly). Без неё программа тоже работает (оприходование, грузоместа, этикетки), но состав заявки автоматически выровнять не сможет — только предупредит.

## 4. Как пользоваться

### Корзина (вверху)
Сначала отберите поставки в работу:
- слева — все ваши активные поставки (номер · кластер · статус);
- нажмите **＋**, чтобы добавить поставку в корзину «В работе» (она уйдёт из левого списка);
- **✕** убирает поставку обратно.

Дальше обе вкладки работают **только по корзине**.

### Вкладка «Оприходование»
Готовит файлы для отправки ФФ **до** сборки заказа.
1. Добавьте нужные поставки в корзину.
2. (по желанию) откройте «Параметры документа» — номер, дата, цена, примечание.
3. Нажмите **«Сформировать оприходования (ZIP)»** — получите архив с XLSX на каждую поставку.
4. Скачайте архив и отправьте файлы в Telegram-группу ФФ.

### Вкладка «Этикетки по описи»
Работает **после** того, как ФФ прислал опись.
1. Выберите поставку из корзины.
2. Загрузите файл-опись (XLSX), который прислал ФФ.
3. Программа сверит номер поставки (зелёная галочка = совпало).
4. Нажмите **«Создать грузоместа и получить этикетки»** — программа создаст грузоместа в Ozon и соберёт **ZIP с этикетками** (каждая названа своим PAC-кодом).
5. Скачайте ZIP и отправьте в Telegram-группу ФФ.

> ⚠️ Кнопка создаёт реальные грузоместа в вашем ЛК Ozon.

## 5. Частые вопросы

**Не вижу новую поставку в списке.**
Программа берёт поставки из Ozon в реальном времени (с обновлением раз в ~2 минуты). Нажмите в боковой панели **«🔄 Обновить данные»**. Показываются поставки в статусах «Заполнение данных» и «Готова к поставке».

**Сайт не открывается.**
Убедитесь, что чёрное окно `start.bat` запущено и не закрыто. Откройте http://localhost:8501 (или http://127.0.0.1:8501). Обновите страницу (Ctrl+F5).

**Куда сохраняются мои ключи и файлы.**
Всё на вашем компьютере, в папке программы. В интернет уходят только запросы напрямую в Ozon.
